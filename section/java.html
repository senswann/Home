Projet JAVA

Présentation

Ce projet JAVA a été mené en binôme au début de ma 2e année.

Contexte

Durant 2 mois, nous avons dû développer en JAVA une application permettant de lire des fichiers .cbz pour pouvoir en faire la lecture.

Développement

Le développement de ce projet m'a permis d'en apprendre plus sur le JAVA et sur la programmation orientée objet (POO) d'une application. Étant en avance lors du développement de ce projet, nous avons décidé de rajouter un système de lecture de musique pendant que l'on lit notre livre.
